Steps on how to create GitHub page: 
1. Go to your repository and click 'Settings'. 
2. Scroll down and change repository visibility to 'Public". 
3. On the left side under Code and Automation, click 'Pages'. 
4. Under 'Branch' (in Build and Deployment section), direct the branch to the one you're using (main). 
5. Wait for GitHub to create a link for your site. 

Key Takeaways on Exercise: 
1. You are always required to initialize (<!DOCTYPE html>) before starting HTML. 
2. Initializing <a> allows you to add links, while (<p>) is for texts. 
3. (<body>) is the tag which contains the content seen by the user.
4. (<head>) contains the title and CSS links you can add to the site.
